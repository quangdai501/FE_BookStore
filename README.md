## Run Locally

Clone the project

```bash
  git clone https://github.com/quangdai501/FE_BookStore.git
```

Go to the project directory

```bash
  cd FE_BookStore
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```


## Build

To Build this project run

```bash
  yarn build
```
